# Fuzzer 🎵

A minimalist soundboard for Windows. Add your own sounds, trigger them instantly, and enjoy a fast, lightweight experience.

## ✨ Features

* **Minimalist Design:** Clean, lightweight, and clutter-free user interface.
* **Custom Sounds:** Easily import and organize your own audio files.
* **Instant Triggers:** Set up hotkeys or quick-click triggers for zero-delay playback.
* **Performance Focused:** Built to run efficiently in the background without hogging Windows resources.

## 🚀 Getting Started

### Installation

1. Head over to the **[Releases](https://github.com/Mellowzyn/Fuzzer/releases)** page.
2. Download the latest version (`Fuzzer v1.0.0`).
3. Extract or run the installer/executable to get started.

### How to Use

1. Launch **Fuzzer**.
2. Click the configuration/add button to import your preferred sound files (e.g., `.mp3`).
3. Assign a preferred hotkey or trigger button to each sound.
4. Let it run in the background and trigger your sounds instantly!
