# Fuzzer

A lightweight Windows soundboard built for simplicity. Load your own clips, assign hotkeys, and play sounds instantly without unnecessary bloat.

## Features

- 🔊 Add and play your own sound clips
- ⌨️ Hotkey support
- ⚡ Lightweight and fast
- 🖥️ Simple, clean interface
- 🚫 No unnecessary features or bloat
